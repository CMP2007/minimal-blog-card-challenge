# Minimalist Blog Card: UI Component

This project is one of my **first** web layout exercises. I've kept it as part of my learning history to show my technical evolution and my foundation in CSS fundamentals.

## 📋 Evolution Note
> ⚠️ **Note:** As this was one of my first projects, the HTML structure and CSS logic (such as the extensive use of `div` containers and fixed values) don't reflect my current level of development. However, it fulfills the main objective of the time: to accurately replicate the visual aesthetic of a modern design.

## 📋 Technical Analysis of the Exercise
In this exercise, I focused on:
- **Visual Fidelity:** Achieving a "Pixel-Perfect" finish compared to the original design.

- **CSS3 Aesthetics:** Implementation of soft shadows (`box-shadow`), rounded corners, and external font management ('Sora').

- **Layout:** Use of Flexbox in the root container for component centering.

## 🛠️ Technologies
- HTML5
- CSS3 (Flexbox and layer effects)
- Google Fonts API
